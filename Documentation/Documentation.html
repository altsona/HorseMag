1: Installation
  1. Unzip the Template.zip file.
  2. On Blogger Dashboard Click Template.
  3. Click on Backup / Restore.
  4. Click on Download Full Template to keep your old template.
  5. Click Browse button. Find where the .xml file located.
  6. Then Select .XML file
  7. Then Click Upload.
  8. Edit mobile Preveiw. (follow images).

2: Mobile Responsive
  1. On Blogger Dashboard Click Template.
  2. Click On Icon.
  3. Click Other.
  4. Check No and Save it
  5. Note:This setting is important to show responsiveness on PC, Mobile etc.


3: Blog Feed
  1. On Blogger Dashboard Click Settings.
  2. Click Other.
  3. In Site Feed > Allow Blog Feed Choose Full.
  4. Then Click Save settings.


4: Search Prefrence
  1. On Blogger Dashboard Click Template.
  2. Click On Search Prefrences Click On Edit
  3. Check YesOption
  4. Write Description Of Blog
  5. Note:Write Upto 150 characters Descripiton about your site


5: Time Stamp Show on Homepage
  1. On Blogger Dashboard Click Language and Formatting.
  2. Click On Timestamp Format and select dd/mm/year
  3. Finally click on Save

6: Display Posts in Category
  1. On Blogger Dashboard Click Layout.
  2. Click Category Lists Edit
  3. Add Exact Label Inside Widget
  4. Click Save.
  5. Note The Label Name must be added also in your Post .


  6. Add Same Label in your Post which you added previously in Widget
  7. Like I added Browsers Label in Widget

  8. Finally click on Save and refresh your page
7: Show Posts on HomePage
  1. To Show 10 Posts on HomePage.
  2. On Blogger Dashboard Click Settings.
  3. Click Basic.
  4. Click Post and Comments
  5. Then Click Save settings.
